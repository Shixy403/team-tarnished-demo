## Overview
Team Tarnished is a 2D action game built using **modular C programming**, where gameplay systems are organized into dedicated modules for readability and maintainability.  
My focus was implementing gameplay mechanics, projectile logic, and level scripting.

## My Contributions
- Implemented `bullet.c` and `bullet.h` for projectile creation, updates, and rendering.  
- Designed player damage, invulnerability, and collision detection systems.  
- Adjusted and refined gameplay elements in `levelTwo.c`, including traps and enemy speeds.  
- Cleaned up warnings, optimized performance, and documented core gameplay modules.

## Controls
- Hold mouse button to charge melee attaack.
- Q to quit at any time during level play.
- X to toggle time stop during play, 1 uses per 12 seconds.
